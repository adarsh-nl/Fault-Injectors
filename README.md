# Griffin Dataset Visualisation

A step-by-step Jupyter notebook for exploring and visualising the Griffin aerial-ground cooperative perception dataset.

## Repository structure

```
griffin-visualisation/
  notebooks/
    griffin_visualisation_tutorial.ipynb   <- main notebook
  docs/
    tutorial_guide.md                      <- detailed guide for every step
    coordinate_transformation.md           <- deep-dive on sensor frame transforms
  requirements.txt
  .gitignore
```

## Quick start

```bash
pip install -r requirements.txt
jupyter notebook notebooks/griffin_visualisation_tutorial.ipynb
```

## Dataset

Griffin is available on Hugging Face: https://huggingface.co/datasets/wjh-svm/Griffin  
Paper: https://arxiv.org/abs/2503.06983  
GitHub: https://github.com/wang-jh18-SVM/Griffin
